Commands

Help

Shows you the list of commands and how they work.

Syntax:
/help

Avatar

Displays your or another user's avatar.

Syntax:

  • To get your own avatar.
  • /avatar

  • To get another user's avatar.
  • /avatar [user]

Warn

Warns a user and gives them an infraction.
Also returns their total number of infractions.

Syntax:

  • To warn a user without a reason.
  • /warn [user]
  • To warn a user with a reason.
  • /warn [user] [reason]

Timeout

Mutes a user for the selected ammount of time (1 minute, 1 hour, 1 day) and gives them an infraction.
Also returns their total number of infractions.

Syntax:

  • To timeout a user without a reason.
  • /timeout [user] [1m/1h/1d]
  • To timeout a user with a reason.
  • /timeout [user] [1m/1h/1d] [reason]

Kick

Kicks a user from the server.

Syntax:

  • To kick a user without a reason.
  • /kick [user]
  • To kick a user with a reason.
  • /kick [user] [reason]

Ban

Bans a user from the server.

Syntax:

  • To ban a user without a reason.
  • /ban [user]
  • To kick a user with a reason.
  • /kick [user] [reason]

Prune

Deletes up to 99 messages from the current channel.

Syntax:
/prune [amount]
